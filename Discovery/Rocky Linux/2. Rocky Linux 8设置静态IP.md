### 传统方法
1. 打开配置文件
```
[root@localhost ~]# vim /etc/sysconfig/network-scripts/ifcfg-ens160
```
2. 修改配置文件
```
BOOTPROTO=static
IPADDR=192.168.178.130
PREFIX=24
GATEWAY=192.168.178.2
DNS1=192.168.178.2
```
3. 如果修改了固定IP需要重启 
注意：
Linux Rocky 8 使用 nmctl 代替 systemctl network
nmctl -c reload = systemctl network reload
